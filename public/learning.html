Learning Goals
  • The Goal of this class is to teach some basic and intermediate concepts behind different languages used in web development such as Java Script, HTML and CSS.



  • This class will cover how to use editing software like Visual Studio Code to compile the code covered throughout the semester.



  • The course will also facilitate a group learning experience by making use of sites such as Haroku.com and Github.com to more realistically portray how group assignments are completed in the field of web design by creating several web pages and then publishing them to a web server.
